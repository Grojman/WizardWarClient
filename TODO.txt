    - CREAR MAZO ALEATORIO (TODAS LAS CARTAS TIENEN EFECTOS RANDOM)
    - CREAR UN SERVICIO PARA EL TEXTO
    - CREAR UN SERVICIO PARA EL ALMACENAMIENTO DE TODAS LAS CARTAS, EFECTOS... ETC
    - CREAR ANIMACIONES PARA LOS FINALES DE PARTIDA (PIERDE, SE VA, EL JUGADOR SE QUEDA SIN CARTAS...)
